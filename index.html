RESEARCH:

So, what did I make?

...When I said I made an AI, did you think of ChatGPT-like AIs?

I don't blame you. It would be really impressive if some random teenager made another Deepseek. (Unfortunately, you'd need a gazillion dollars to even train the thing.)

But no, I didn't make a chatbot like ChatGPT; I made a Convolutional Neural Network.

In the world of Machine Learning (where algorithms do tasks without explicit programming to do so), the most popular 'archetype' of the AI is the Neural Network.

Essentially, it is an algorithm designed to mimic a simplified brain using basic processing units, called 'Neurons'.

Both ChatGPT-likes and Convolutional Neural Networks are modified types of the Neural Network:

ChatGPT is an LLM (Large Language Model) that is built off Transformer Neural Network, specifically designed to process language data. On the other hand, CNNs (Convolutional Neural Networks) are the architecture that build Image Processing Models such as YOLO or AlexNet, finding patterns within the images.

STRUCTURE:

How does it work?

To understand how my CNN would work, we first have to understand how any Neural Network works:

Simply put, a Neural Network is an arrangement of basic processing units, 'Neurons', into layers in order to approximate a function; it defines a function without the definition of that function.

If you imagine a graph as a string that bends and twists, a Neural Network would be an approximation of that string, with the Neurons doing the bending and twisting.

As previously stated, a CNN is a modified Neural Network. The only major difference is that its made up of three layers, two of which (CL and FCL) being standalone Neural Networks. This is shown in the diagram I made on the right.

As seen here in the diagram below, a CNN is comprised of three layers: CL, Pooling, and FCL.

The CL is a modified Neural Network, where each neuron can't see of the previous layer; this layer specializes in feature extraction.

The Pooling Layer compresses the 3D Tensor for better optimization.

The FCL is a normal Neural Network where all of the previous layer is visible. It's better for classification or detection.
