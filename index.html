RESEARCH:

So, what did I make?

...When I said I made an AI, did you think of ChatGPT-like AIs?

I don't blame you. It would be really impressive if some random teenager made another Deepseek. (Unfortunately, you'd need a gazillion dollars to even train the thing.)

But no, I didn't make a chatbot like ChatGPT; I made a Convolutional Neural Network.

In the world of Machine Learning (where algorithms do tasks without explicit programming to do so), the most popular 'archetype' of the AI is the Neural Network.

Essentially, it is an algorithm designed to mimic a simplified brain using basic processing units, called 'Neurons'.

Both ChatGPT-likes and Convolutional Neural Networks are modified types of the Neural Network:

ChatGPT is an LLM (Large Language Model) that is built off Transformer Neural Network, specifically designed to process language data. On the other hand, CNNs (Convolutional Neural Networks) are the architecture that build Image Processing Models such as YOLO or AlexNet, finding patterns within the images.

"A neural network is an algorithm that can do machine learning (approximate a function without explicitly writing the definition of the function; when a computer teaches itself to do something). A neural network does this by using small processing units called ‘Neurons’, inspired by the real-life neurons found within the brain. ‘Neurons’ are organized in multiple layers, starting with an input layer, through ‘hidden’ layers, then finally to an output layer. The process of running a neural network uses Forward Propagation (where data is passed forward layer-by-layer), and the process of training/teaching the neural network uses Back Propagation (where the output of the algorithm is measured to see how much to change the biases of the neurons) (Fangfang). Neural networks are useful in situations where it is difficult to set a concrete, mathematical definition, such as facial detection.

STRUCTURE:

How does it work?

To understand how my CNN would work, we first have to understand how any Neural Network works:

Simply put, a Neural Network is an arrangement of basic processing units, 'Neurons', into layers in order to approximate a function; it defines a function without the definition of that function.

As previously stated, a CNN is a modified Neural Network. The only major difference is that its made up of three layers, two of which (Convolutional Layer and Fully-Connected Layer) being standalone Neural Networks. This is shown in the diagram on the right (only seen on desktop version).

As seen here in the diagram below, a CNN is comprised of three layers: CL, Pooling, and FCL.

The CL is a modified Neural Network, where each neuron can't see of the previous layer; this layer specializes in feature extraction.

The Pooling Layer compresses the 3D Tensor for better optimization.

The FCL is a normal Neural Network where all of the previous layer is visible. It's better for classification or detection.

In the end, I successfully met my criteria (...ish, at the time of writing I've written working structure but I haven't actually tested it altogether). Either way, I learned multiple things through this project, each one making the project worthwhile.

Finally, I required creativity to design the algorithm and to be able to put the building blocks (functions, neurons, etc.) together.

To conclude, this project has tested me, pushing my creative, problem-solving, and mathematical skills. My takeaway: the difficult aspect to programming is not the code itself, nor the design of the algorithm, but instead the way the code forces you to to think about the most miniscule details, and how they add up to the big picture.
